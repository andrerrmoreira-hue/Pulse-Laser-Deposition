# Pulse Laser Deposition (PLD)

A custom-built **Pulsed Laser Deposition (PLD)** platform developed at the **Instituto Superior de Engenharia de Lisboa (ISEL)** for the fabrication of thin films and nanostructured materials for photonic and optoelectronic applications.

The system was designed as a flexible research platform, integrating custom mechanical components, electronics, and software to provide precise control over the deposition process while remaining easily expandable for future developments.

---

## Features

* Custom-designed PLD deposition chamber
* Multi-target deposition system
* Heated substrate holder
* Precision mechanical positioning
* Custom electronics and matching network
* Modular mechanical design
* Fully designed using SolidWorks CAD

## Mechanical Design

The complete PLD platform was designed in SolidWorks.

The repository includes:

* Complete assemblies
* Individual mechanical parts
* Target holder
* Substrate carrier
* Chamber components
* Heating system supports
* Motion mechanisms
* Custom fixtures

These files allow the system to be reproduced, modified, or further improved.
